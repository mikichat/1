# 📊 엑셀 자동 입력 기능 가이드

## 🎯 개요

엑셀 파일(.xlsx, .xls)을 업로드하면 여행 정보를 자동으로 추출하여 입력 폼을 채워주는 기능입니다.

**수작업으로 입력하던 시간을 90% 절약!**

---

## ✨ 주요 기능

### 1. 🚀 빠른 입력
- 엑셀 파일 하나로 모든 정보 입력
- 드래그 앤 드롭 지원
- 자동 데이터 매칭

### 2. 📋 지원 형식
- `.xlsx` (Excel 2007 이상)
- `.xls` (Excel 97-2003)

### 3. 🔍 자동 추출 항목
- 여행 제목
- 출발일/도착일
- 출발공항/도착공항
- 숙소 정보
- 회사 정보
- 담당자 정보

### 4. ✏️ 수동 수정 가능
- 자동 입력 후 필요시 수정
- 누락된 정보 추가 입력
- 일반 플로우대로 진행

---

## 🛠️ 사용 방법

### Step 1: 엑셀 파일 업로드

1. **홈페이지에서 "📊 엑셀 파일로 자동 입력" 버튼 클릭**
   ```
   index.html → "엑셀 파일로 자동 입력" 클릭
   ```

2. **파일 업로드 (2가지 방법)**
   - **방법 A**: 파일 선택 버튼 클릭
   - **방법 B**: 엑셀 파일을 드래그 앤 드롭

3. **지원 형식 확인**
   - ✅ .xlsx (Excel 2007+)
   - ✅ .xls (Excel 97-2003)
   - ❌ .csv는 미지원

---

### Step 2: 데이터 미리보기

업로드 후 자동으로 추출된 데이터를 확인할 수 있습니다.

```
┌─────────────────────────────┐
│ 📋 추출된 데이터 미리보기    │
├─────────────────────────────┤
│ 여행 제목: 다낭 골프 여행    │
│ 출발일: 2024-11-12          │
│ 도착일: 2024-11-15          │
│ 출발 공항: 인천국제공항      │
│ 도착 공항: 다낭국제공항      │
│ 숙소: 그랜드 호텔            │
│ 회사명: (주)여행세상         │
│ 담당자: 홍길동               │
│ 담당자 전화: 010-1234-5678  │
└─────────────────────────────┘
```

**확인 사항:**
- ✅ 데이터가 올바르게 추출되었는가?
- ✅ 날짜 형식이 정확한가?
- ✅ 누락된 정보는 없는가?

---

### Step 3: 입력 폼에 적용

1. **"이 데이터로 입력 폼 채우기" 버튼 클릭**
2. 자동으로 `golf-advanced.html`로 이동
3. 추출된 데이터가 자동으로 입력됨

**자동 입력되는 필드:**
- ✅ 타이틀
- ✅ 여행 기간 (시작일/종료일)
- ✅ 공항 미팅 정보
- ✅ 현지 미팅 정보
- ✅ 항공편 정보
- ✅ 숙소 정보
- ✅ 회사 정보
- ✅ 담당자 정보

---

### Step 4: 수동 수정 및 추가

자동 입력 후 다음 작업을 진행하세요:

1. **누락된 정보 추가**
   - 골프장 & TEE-UP 정보
   - 일정 및 식사
   - 이미지 업로드

2. **잘못 입력된 정보 수정**
   - 날짜 형식 확인
   - 전화번호 형식 확인

3. **디자인 설정**
   - 폰트 선택
   - 색상 설정
   - 템플릿 적용

---

### Step 5: 미리보기 및 저장

1. **"미리보기 & 공유하기" 클릭**
2. 미리보기에서 최종 확인
3. 필요시 "수정하기"로 돌아가서 조정
4. 완료 후 저장 또는 공유

---

## 📝 엑셀 파일 형식 가이드

### 권장 형식

엑셀 파일은 다음과 같은 구조로 작성하는 것을 권장합니다:

```
| 항목           | 값                    |
|----------------|-----------------------|
| 여행명         | 다낭 골프 여행        |
| 출발일         | 2024-11-12           |
| 도착일         | 2024-11-15           |
| 출발공항       | 인천국제공항          |
| 도착공항       | 다낭국제공항          |
| 숙소           | 그랜드 호텔           |
| 회사명         | (주)여행세상          |
| 담당자 이름    | 홍길동                |
| 담당자 전화    | 010-1234-5678        |
| 담당자 이메일  | hong@company.com     |
```

### 키워드 매칭 규칙

시스템이 자동으로 인식하는 키워드:

| 키워드              | 매칭되는 필드         |
|---------------------|----------------------|
| 여행명, 제목        | 타이틀               |
| 출발일, 시작일      | 시작일               |
| 도착일, 종료일      | 종료일               |
| 출발공항            | 출발 공항            |
| 도착공항            | 도착 공항            |
| 숙소 (주소 제외)    | 숙소명               |
| 회사명              | 회사명               |
| 담당자 이름         | 담당자명             |
| 담당자 전화         | 담당자 전화번호      |
| 담당자 이메일       | 담당자 이메일        |

**💡 Tip:** 키워드는 부분 매칭됩니다. 예: "출발일자" → "출발일"로 인식

---

### 날짜 형식

다음 형식의 날짜를 자동으로 인식합니다:

✅ **지원하는 형식:**
- `2024-11-12` (ISO 형식)
- `2024.11.12` (점 구분)
- `2024/11/12` (슬래시 구분)
- `20241112` (숫자만)
- 엑셀 날짜 (숫자 형식)

❌ **지원하지 않는 형식:**
- `11월 12일` (한글)
- `Nov 12` (영문)

**💡 Tip:** 엑셀에서 날짜를 `YYYY-MM-DD` 형식으로 지정하면 가장 정확합니다.

## ⚠️ 주의사항

### 1. 첫 번째 시트만 읽음
- 엑셀 파일의 첫 번째 시트만 읽습니다
- 여러 시트가 있어도 첫 번째만 처리

### 2. 키워드 기반 매칭
- 정확한 키워드가 없으면 추출되지 않음
- 권장 형식을 따르는 것이 가장 정확

### 3. 일부 정보는 수동 입력 필요
- 골프장 & TEE-UP 정보
- 일정 및 식사 상세
- 이미지 파일

### 4. 데이터 검증 필요
- 자동 추출 후 반드시 확인
- 날짜, 전화번호 형식 체크
- 누락된 정보 보완

## 🔧 문제 해결

### Q1: 엑셀 파일이 업로드되지 않아요
**A:** 다음을 확인하세요:
- 파일 형식이 `.xlsx` 또는 `.xls`인가?
- 파일 크기가 10MB 이하인가?
- 파일이 손상되지 않았나?

### Q2: 데이터가 추출되지 않아요
**A:** 다음을 확인하세요:
- 첫 번째 시트에 데이터가 있나?
- 키워드가 정확한가?
- 날짜 형식이 올바른가?

### Q3: 일부만 추출되어요
**A:** 정상입니다!
- 키워드가 매칭된 항목만 추출
- 나머지는 수동으로 입력
- 예제 파일 형식 참고

### Q4: 날짜가 이상하게 나와요
**A:** 엑셀 날짜 형식 문제일 수 있습니다:
- 엑셀에서 날짜를 `YYYY-MM-DD` 텍스트로 변환
- 또는 업로드 후 수동으로 수정

## 💡 활용 팁

### Tip 1: 엑셀 템플릿 만들기
자주 사용하는 형식의 엑셀 템플릿을 만들어두면 편리합니다.

```
template.xlsx 저장
↓
정보만 변경
↓
업로드
↓
자동 입력!
```

### Tip 2: 회사 정보 미리 입력
회사 정보는 매번 동일하므로 템플릿에 미리 입력해두세요.

### Tip 3: 단계별 완성
1차: 엑셀로 기본 정보 입력
2차: 수동으로 상세 정보 추가
3차: 디자인 적용

### Tip 4: 예제 파일 활용
예제 파일을 다운로드하여 수정하면 형식 오류를 방지할 수 있습니다.
