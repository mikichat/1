# ✨ 리치 텍스트 에디터 사용법

## 📝 개요

이제 **추가 안내사항**, **일정 상세**, **식사 메뉴** 입력 시 텍스트를 블록 지정하여 **글자 색상**, **굵기**, **기울임** 등을 자유롭게 변경할 수 있습니다!

---

## 🎯 적용 대상

### 1. 추가 안내사항
- 준비물, 주의사항 등 자유롭게 작성

### 2. 일정 상세 (각 일정마다)
- 시간별 일정, 이동 경로 등

### 3. 식사 메뉴 및 포함 사항 (각 일정마다)
- 조식/중식/석식 메뉴
- 포함/미포함 사항

---

## 🛠️ 사용 방법

### 1️⃣ 기본 텍스트 입력
1. 에디터 영역을 클릭
2. 텍스트 입력 (일반 입력과 동일)

### 2️⃣ 텍스트 서식 변경
1. **색상을 변경하고 싶은 텍스트를 드래그**하여 블록 지정
2. 상단 도구 모음에서 원하는 서식 선택:

#### 📌 사용 가능한 서식

| 아이콘 | 기능 | 설명 |
|--------|------|------|
| **B** | 굵게 (Bold) | 텍스트를 굵게 표시 |
| *I* | 기울임 (Italic) | 텍스트를 기울임 |
| <u>U</u> | 밑줄 (Underline) | 텍스트에 밑줄 추가 |
| ~~S~~ | 취소선 (Strike) | 텍스트에 취소선 추가 |
| 🎨 A | 글자 색상 | 텍스트 색상 변경 |
| 🎨 ▮ | 배경색 | 텍스트 배경색 변경 |
| ≡ | 정렬 | 왼쪽/가운데/오른쪽 정렬 |
| 1. | 번호 목록 | 번호 매기기 목록 |
| • | 글머리 목록 | 점 글머리 목록 |
| 🧹 | 서식 지우기 | 모든 서식 제거 |

---

## 🎨 글자 색상 변경 방법

### 방법 1: 미리 정의된 색상 사용
1. 텍스트 드래그하여 블록 지정
2. 도구 모음에서 **A** (글자 색상) 버튼 클릭
3. 색상 팔레트에서 원하는 색상 선택

### 방법 2: 사용자 정의 색상
1. 텍스트 드래그하여 블록 지정
2. 도구 모음에서 **A** (글자 색상) 버튼 클릭
3. 색상 팔레트 하단의 **"사용자 정의"** 클릭
4. 색상 선택기에서 원하는 색상 선택

## 💡 활용 예시

### 예시 1: 중요 사항 강조
```
준비물:
- 여권 (필수) ← 빨간색, 굵게
- 골프화
- 선글라스
```

**적용 방법:**
1. "(필수)" 텍스트 드래그
2. **B** (굵게) 클릭
3. **A** (글자 색상) 클릭 → 빨강 선택

### 예시 2: 식사 메뉴 구분
```
조식: 호텔 뷔페 (포함) ← 파랑
중식: 한정식 (포함) ← 파랑
석식: 자유식 (미포함) ← 빨강
```

**적용 방법:**
1. "(포함)" 텍스트 드래그 → 파란색 적용
2. "(미포함)" 텍스트 드래그 → 빨간색 적용

### 예시 3: 일정 강조
```
08:30 호텔 조식
09:00 티업 ← 굵게, 녹색
13:00 중식
```

**적용 방법:**
1. "09:00 티업" 드래그
2. **B** (굵게) + 녹색 적용

## 📋 목록 만들기

### 번호 목록 (1, 2, 3...)
1. 도구 모음에서 **1.** 버튼 클릭
2. 텍스트 입력 후 Enter
3. 자동으로 번호가 증가

### 글머리 목록 (•)
1. 도구 모음에서 **•** 버튼 클릭
2. 텍스트 입력 후 Enter
3. 자동으로 글머리 추가

## 🔍 미리보기에서 확인

1. 서식을 적용한 후 "미리보기 & 공유하기" 클릭
2. 미리보기 페이지에서 색상, 굵기 등이 **그대로 표시됨**
3. "고해상도 이미지 저장" 시에도 **모든 서식 포함**

---

## ⚠️ 주의사항

### 1. 서식 누적
- 같은 텍스트에 여러 서식 중첩 가능
- 예: 빨강 + 굵게 + 밑줄

### 2. 서식 제거
- 텍스트 드래그 후 🧹 (서식 지우기) 클릭
- 또는 서식 버튼 다시 클릭하여 토글

### 3. 복사/붙여넣기
- 다른 프로그램에서 복사한 텍스트는 서식이 제거됨
- 에디터 내에서 복사/붙여넣기는 서식 유지

## 🎯 실전 활용 팁

### Tip 1: 색상 일관성
- 중요: 빨강
- 포함: 파랑
- 미포함: 주황
- 추천: 녹색

### Tip 2: 가독성
- 너무 많은 색상 사용 자제
- 굵기로 먼저 강조, 색상은 보조

### Tip 3: 목록 활용
- 준비물 → 글머리 목록
- 일정 순서 → 번호 목록

## 📱 스마트폰에서도 적용됨

- ✅ 모든 색상 유지
- ✅ 굵기, 기울임 유지
- ✅ 목록 형식 유지
- ✅ 배경색 유지

## 🆘 문제 해결

### Q: 색상이 표시되지 않아요
**A:** 텍스트를 먼저 드래그하여 블록 지정했는지 확인하세요.

### Q: 서식이 저장되지 않아요
**A:** "미리보기 & 공유하기" 전에 에디터 외부를 한 번 클릭하세요.

### Q: 기존 데이터 불러오기 시 서식이 사라져요
**A:** 기존에 일반 텍스트로 저장된 데이터는 서식이 없습니다. 다시 서식을 적용하고 저장하세요.

### Q: 미리보기에서 깨져 보여요
**A:** 페이지를 새로고침하고 다시 확인해보세요.
