## 🎨 사용 가능한 기능

- ✅ 번호 목록 (1, 2, 3...)

### 제목
- ✅ 제목 1 (큰 제목)
- ✅ 제목 2 (중간 제목)

### 기타

## 📱 데이터 저장 및 로드

### 저장 방식
1. Quill 에디터의 HTML 콘텐츠를 hidden input에 자동 동기화
2. 폼 제출 시 HTML 그대로 저장
3. localStorage 및 URL 압축 지원

### 불러오기 방식
1. 저장된 HTML 콘텐츠를 Quill 에디터에 로드
2. 모든 서식 (색상, 굵기, 목록 등) 완벽 복원

### 미리보기
1. HTML 콘텐츠를 그대로 렌더링
2. Quill CSS 스타일 적용으로 동일한 모양 유지

### 이미지 저장
1. html2canvas로 캡처 시 모든 서식 포함
2. 고해상도 (3배) PNG로 저장

## 🧪 테스트 방법

### 1. 기본 텍스트 입력 테스트
```
1. golf-advanced.html 열기
2. "추가 안내사항" 에디터 클릭
3. 텍스트 입력
4. 정상 입력 확인 ✅
```

### 2. 색상 변경 테스트
```
1. 텍스트 입력: "여권 필수"
2. "필수" 텍스트 드래그
3. 도구 모음에서 A (색상) 클릭
4. 빨강 선택
5. 빨간색으로 변경 확인 ✅
```

### 3. 여러 서식 적용 테스트
```
1. 텍스트 입력: "중요 안내"
2. "중요 안내" 드래그
3. B (굵게) 클릭
4. A (색상) → 파랑 선택
5. U (밑줄) 클릭
6. 굵은 파란색 밑줄 텍스트 확인 ✅
```

### 4. 저장/불러오기 테스트
```
1. 서식 적용 후 "현재 안내문 저장"
2. 페이지 새로고침
3. "저장된 안내문 불러오기"
4. 모든 서식 복원 확인 ✅
```

### 5. 미리보기 테스트
```
1. 서식 적용 후 "미리보기 & 공유하기"
2. 미리보기 페이지에서 서식 표시 확인 ✅
3. "고해상도 이미지 저장"
4. 다운로드된 이미지에서 서식 확인 ✅
```

### 6. 일정 추가 테스트
```
1. "일정 추가" 버튼 클릭
2. 새로 추가된 일정의 에디터 확인
3. 텍스트 입력 및 서식 적용
4. 정상 작동 확인 ✅
```

## ⚠️ 주의사항

### 1. 에디터 높이
- 최소 높이: 120px (일정), 150px (추가 안내사항)
- 내용이 많아지면 자동으로 높이 증가

### 2. 이미지 삽입
- 현재 에디터 내 이미지 삽입 기능 없음
- 별도 이미지 업로드 필드 사용

### 3. 하위 호환성
- 기존 일반 텍스트 데이터도 정상 표시
- 서식 없이 표시되며, 다시 저장 시 HTML로 변환

### 4. 브라우저 호환성
- 최신 Chrome, Firefox, Safari, Edge 지원
- IE는 지원하지 않음

## 🔧 기술 정보

### Quill.js v2.0.2
- 공식 사이트: https://quilljs.com/
- CDN: jsdelivr
- 테마: Snow (깔끔한 UI)

### 도구 모음 구성
```javascript
toolbar: [
  [{ 'header': [1, 2, 3, false] }],      // 제목
  ['bold', 'italic', 'underline', 'strike'],  // 서식
  [{ 'color': [] }, { 'background': [] }],    // 색상
  [{ 'align': [] }],                          // 정렬
  [{ 'list': 'ordered'}, { 'list': 'bullet' }], // 목록
  ['clean']                                   // 서식 지우기
]
```

### 데이터 구조
```javascript
{
  notes: "<p>텍스트 <strong style='color: red;'>빨강</strong></p>",
  schedules: [
    {
      detail: "<p><strong>09:00</strong> 티업</p>",
      meals: "<p><span style='color: blue;'>포함</span></p>"
    }
  ]
}
```
