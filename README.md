# 🏌️✈️ 여행 안내문 생성기

골프 여행과 일반 여행을 위한 전문적인 안내문을 쉽고 빠르게 작성할 수 있는 웹 기반 시스템입니다.

## 🎯 프로젝트 개요

여행사나 단체 여행을 준비하는 분들을 위한 프리미엄 안내문 작성 도구입니다. 

**2가지 버전을 제공합니다:**
1. **⛳ 골프 여행 안내문** - 골프장 & TEE-UP 중심
2. **✈️ 일반 여행 안내문** - 일정표 중심

**서버 기반 저장 시스템:**
- Python 백엔드 서버 (SQLite 데이터베이스)
- 안내문 및 템플릿 영구 저장
- 원클릭 실행 배치 파일 (start.bat)
- Tailwind CSS 자동 빌드

### 🧪 사용 방법

#### 서버 실행
1. **자동 실행 (권장)**
   ```batch
   start.bat
   ```
   - Python 및 Node.js 설치 확인
   - Tailwind CSS 빌드 자동 시작
   - Python 서버 자동 시작
   - 브라우저에서 `http://localhost:8000` 접속

2. **수동 실행**
   ```bash
   # Tailwind CSS 빌드 (별도 터미널)
   npx @tailwindcss/cli -i ./css/input.css -o ./css/output.css --watch
   
   # Python 서버 시작
   python server.py
   ```

#### 안내문 작성
1. **입력 페이지** (`golf-advanced.html`)
   - 타이틀 입력 (필수)
   - 여행 기간 선택
   - 공항 미팅 / 현지 미팅 정보 입력
   - 골프장 추가 (무제한)
   - 일정 추가 (무제한)
   - "미리보기 & 공유하기" 클릭

2. **미리보기 페이지** (자동 이동)
   - 입력한 모든 정보 확인
   - 인쇄하기
   - 링크 복사 (다른 사람과 공유)
   - 수정하기 (입력 페이지로 돌아가기)

3. **저장 및 불러오기 (서버 기반)**
   - "현재 안내문 저장" - 서버에 영구 저장
   - "저장된 안내문 불러오기" - 서버에서 불러오기
   - "현재 디자인을 템플릿으로 저장" - 디자인만 서버에 저장
   - "템플릿 불러오기" - 서버에서 디자인만 적용

4. **엑셀 자동 입력**
   - 템플릿 다운로드: `엑셀_템플릿_생성.html` → 빈 템플릿 받기
   - 예제 확인: `엑셀_예제파일_생성.html` → 다낭 3박4일 예제 파일
   - 엑셀 작성: 템플릿 형식에 맞춰 데이터 입력
   - 업로드: `excel-import.html` → 엑셀 파일 드래그 또는 선택
   - 자동 변환: 추출된 데이터 확인 후 "입력 폼 채우기" 클릭

## ✨ 프리미엄 기능 (양쪽 모두 동일)

### 📸 이미지 시스템
- 타이틀 배경 이미지
- 공항/현지 미팅 이미지 (각각 별도)
- 항공편 정보 이미지
- 골프장별 개별 이미지
- 일정별 개별 이미지
- 회사 로고 추가
- 이미지 크기 조절 (50px ~ 800px)
- Base64 인코딩으로 저장

### 🎨 디자인 커스터마이징
- **8가지 한글 폰트 선택**
  - Pretendard (ExtraBold, Light, Medium, Regular)
  - 레시피코리아
  - SB 어그로 (B, L, M)
  - Noto Sans, 나눔고딕, 나눔명조 등
- **완전한 색상 커스터마이징**
  - 헤더 배경색
  - 타이틀 색상
  - 섹션별 배경색 (미팅, 항공편, TEE-UP, 일정, 숙소, 안내사항)
  - 테두리 색상
- **자동 여백 조절** (컨텐츠 양에 따라 최적화)

### 💾 서버 기반 저장 시스템
- **데이터베이스 저장**
  - SQLite 데이터베이스 사용
  - 영구적인 데이터 저장
  - 브라우저 제약 없음 (localStorage 용량 제한 해결)
- **안내문 관리**
  - 여러 개의 안내문을 이름으로 구분하여 저장
  - 저장된 목록에서 선택하여 불러오기
  - 저장 시간 자동 기록
- **템플릿 시스템**
  - 현재 디자인 설정을 템플릿으로 저장
  - 템플릿 불러오기 (텍스트 유지, 디자인만 적용)
  - 템플릿 삭제 가능

### 📱 공유 기능
- 링크 복사 (URL로 데이터 전달)
- 고해상도 이미지 저장 (3배 해상도)
- 인쇄 기능

## 🔧 기술 스택

### 프론트엔드
- **HTML5** - 시맨틱 마크업
- **CSS3** - 그라디언트, Flexbox, 반응형 디자인
- **JavaScript (Vanilla)** - 동적 기능 구현
- **Tailwind CSS** - 유틸리티 퍼스트 CSS 프레임워크
- **LZ-String** - 데이터 압축 라이브러리 (URL 공유)
- **html2canvas** - 고해상도 이미지 저장 (3x scale)
- **Quill.js** - 리치 텍스트 에디터
- **SheetJS (xlsx)** - 엑셀 파일 읽기 및 파싱
- **FontAwesome** - 아이콘 라이브러리
- **Google Fonts & 커스텀 폰트** - 다양한 한글 폰트

### 백엔드
- **Python 3** - HTTP 서버 구현
- **http.server** - 내장 HTTP 서버 모듈
- **SQLite3** - 경량 데이터베이스
- **JSON** - 데이터 직렬화

### 개발 도구
- **Node.js & npm** - Tailwind CSS 빌드
- **Batch Script** - Windows 자동화
